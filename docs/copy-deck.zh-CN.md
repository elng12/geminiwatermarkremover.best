## FAQ

### Is this tool free?

`Yes. This version is free to use, with no sign-up and no daily credit system.`

### Do I need to sign up?

`No. You can start without creating an account.`

### Does it work on mobile?

`Yes. You can upload, preview, compare, and download from your phone.`

### What file types are supported?

`JPG, PNG, and WebP are supported in this version.`

### Will my image be uploaded?

`No. In this version, the cleanup runs entirely in your browser.`

### What if the automatic removal misses the mark?

`Use manual selection to mark the visible watermark area and try again.`

### Does this remove Gemini logos and icons too?

`It is designed for the visible Gemini sparkle watermark and similar small visible corner marks. Results may vary on other overlays.`

### Does this remove SynthID or invisible watermarks?

`No. This tool is for visible Gemini marks only. It does not remove SynthID, invisible watermarks, or provenance metadata.`

### Does it work for Nano Banana images?

`It can help when the image contains the same visible Gemini-style sparkle mark. It does not remove SynthID, invisible watermarks, or provenance metadata.`

### Does it remove video watermarks?

`No. This version is for images only.`

### Is there a Chrome extension?

`You do not need to install any extension. The tool works directly in your web browser.`

### Does it support bulk or batch removal?

`Currently, this tool processes images one by one. Bulk or batch removal might be added in the future.`

### Do I need to download an app or software for Mac/Windows?

`No software download is required. It runs 100% directly in your browser.`

### Will the image lose quality (is it lossless)?

`The tool processes only the marked area locally, keeping the rest of your original image intact and preserving its quality.`
