## 已完成决策

以下内容在 Sprint 0 内**不应被重新打开反复讨论**：

1. **产品范围已定**
   - 只做图片，不做视频
   - 只做单张，不做批量
   - 不做账号、订阅、API、浏览器扩展

2. **技术方向已定**
   - 纯浏览器处理
   - 主路径使用 `reverse alpha blending`
   - 补充路径使用 `OpenCV.js`
   - 自动检测失败时提供手动选区重试

3. **页面结构已定**
   - 只保留 `/`
   - 保留 `/privacy-policy`
   - 保留 `/terms-of-service`
   - `/contact` 可后补

4. **SEO 原则已定**
   - 一个强首页承接主要关键词
   - 不再拆 `free`、`online`、`logo remover` 等相似页面
   - `Nano Banana` 先由首页 FAQ 自然承接

5. **隐私边界已定**
   - 图片不上传服务器
   - v1 不存在云端兜底
   - 后端不接收用户图片内容
