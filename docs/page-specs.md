# Page Specs

## Route Plan

This product uses one SEO-first homepage plus legal pages.

Required routes:

- `/`
- `/privacy-policy`
- `/terms-of-service`

Optional later route:

- `/contact`

Do not create extra landing pages for small keyword variations.

## Homepage `/`

### Purpose

The homepage must do four jobs at once:

1. rank for the main search demand
2. convert users into upload starts
3. explain the pure in-browser privacy model
4. set realistic expectations about visible-only cleanup

### Primary Query Coverage

Cover these phrases naturally:

- gemini watermark remover online
- gemini watermark remover
- remove gemini watermark
- remove gemini watermark from image
- gemini logo remover
- gemini icon remover
- gemini mark remover
- free online gemini watermark remover
- no sign up gemini watermark remover
- nano banana watermark remover

Do not stuff them repeatedly in one paragraph.
Map them to sections and FAQ.

### Required Section Order

#### 1. Hero + Tool

Goals:

- explain value in under 5 seconds
- let the user upload immediately
- show free, no sign-up, and in-browser trust signals

Must include:

- main headline
- supporting headline
- upload box
- file format hint
- no-upload-required line
- in-browser line
- `Try an example` entry

#### 2. Trust Signals

Must include 4 short badges:

- Free online
- No sign-up
- 100% in your browser
- No upload required

#### 3. Example Gallery

Show 2 to 3 demo scenarios.

Each card should show:

- before thumbnail
- mark type label
- one-click try button

#### 4. Before/After Proof

Use a comparison slider on desktop and mobile.

Do not use a tiny side-by-side layout as the only mobile comparison method.

#### 5. How It Works

Use a 3-step explanation:

1. Upload or try an example
2. Detect and remove the visible mark in your browser
3. Preview and download the cleaned image

#### 6. What It Works Best For

Explain supported cases:

- small corner sparkle watermark
- gemini-style logo or icon
- simple visible mark on low-detail areas
- images where the visible mark stays near the bottom-right corner

#### 7. Limits and Honest Boundaries

Explain unsupported or unstable cases:

- large overlays
- marks over faces or dense texture
- heavily cropped or severely recompressed images
- invisible watermark
- SynthID
- provenance metadata
- video

#### 8. FAQ

Must answer:

- free or not
- sign-up required or not
- supported file types
- mobile support
- whether the image is uploaded
- what happens if auto-detection misses the mark
- SynthID question
- video question
- Nano Banana question

#### 9. Privacy + Responsible Use

Short trust block near the bottom:

- images are processed in the browser
- cleanup does not require server upload
- use only on images you own or are allowed to edit
- this tool only targets visible Gemini-style marks

#### 10. Footer

Must include:

- trademark disclaimer
- links to legal pages
- placeholder contact email

### Metadata Rules

- unique title
- unique meta description
- one clear H1
- FAQ schema only if the FAQ content is visibly rendered on the page
- OG image should show the tool UI or before/after proof

### Mobile Rules

- upload box visible without much scrolling
- comparison slider works on touch
- download button stays obvious
- manual retry controls are readable and easy to tap

## Privacy Policy `/privacy-policy`

### Purpose

Explain data handling in plain English.

### Must Cover

- browser-only image processing in v1
- no image upload in v1
- any local browser storage or temporary state if used
- GA4 analytics use
- contact channel for privacy requests

## Terms of Service `/terms-of-service`

### Purpose

Set use boundaries and disclaimers.

### Must Cover

- use only on content you own or have permission to edit
- no illegal or abusive use
- service may change or be unavailable
- visible-mark-only scope
- no promise of perfect results
- complaint and legal contact
